1. Install lerna, use the following command for the same
   npm install --global lerna
2. Run command "lerna bootstrap", it'll install all the node_modules in the respective packages.
3. For now we'll have to use 3 tabs to watch both web and native app.
4. Go inside "packages/share" folder and run "npm run watch"
5. Go inside "packages/web" folder and run "npm run start"
6. Go inside "packages/native" folder and run "expo start"
